# Italian Learning App

A dynamic React application built with Vite for learning Italian. Features include vocabulary flashcards, grammar reference, and interactive quizzes (multiple-choice and open-ended questions) to test your knowledge.

## Prerequisites

Before running the project, make sure you have [Node.js](https://nodejs.org/) installed on your machine.

## Installation & Setup

1. **Clone the repository** (if you haven't already) and navigate to the project folder:
   ```bash
   # navigate to the app directory if needed
   cd app
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start the development server**:
   ```bash
   npm run dev
   ```

4. **Open in Browser**:
   Once the server starts, open your browser and navigate to the local URL provided in the terminal (usually `http://localhost:5173`).

## Available Scripts

In the project directory, you can run:

- `npm run dev` - Starts the development server.
- `npm run build` - Builds the app for production to the `dist` folder.
- `npm run preview` - Boots up a local static web server that serves the files from `dist` for previewing the production build.
- `npm run lint` - Runs ESLint to catch potential errors in the code.

## Tech Stack

- **React 19**
- **Vite**
- **react-markdown** & **remark-gfm** (for rendering markdown content)
